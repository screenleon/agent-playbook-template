# Design To Code

Use this skill for image-led UI implementation.

## Required process

Before coding, break down:

1. layout
2. typography
3. color
4. spacing
5. radius, border, shadow
6. interaction states
7. responsive or device differences

## Reporting

At the end, explicitly separate:

- exact matches to the provided design
- reasonable inferences where the design source is incomplete

## Do not

- redesign the screen without being asked
- abstract too early
- replace a specific visual source with a generic template
